# IP地址管理系统 v2.0

一个功能完整的IP地址管理工具，支持子网管理、IP分配、状态跟踪和统计报告。

## 功能特点

### 1. 子网管理
- 支持CIDR格式子网创建（如：192.168.1.0/24）
- 自动计算子网大小和可用IP范围
- 子网状态标识（正常、高使用率、空闲、已满）
- 子网使用率统计和进度显示

### 2. IP地址管理
- IP地址分配、释放、保留
- 批量IP分配功能
- IP状态颜色标识（绿色=空闲，红色=已用，黄色=保留）
- 支持MAC地址和设备类型记录
- IP地址按升序排列

### 3. 搜索和筛选
- 快速搜索（IP地址、主机名、MAC地址）
- 高级搜索（多条件组合）
- 按子网、状态、设备类型筛选
- 实时显示搜索结果

### 4. 导入导出功能
- 导出所有数据（CSV格式）
- 导出子网数据
- 导出搜索结果
- 导入子网数据
- 导入IP数据
- 支持Excel兼容的UTF-8-BOM编码

### 5. 统计和报告
- 全局IP使用率统计
- 子网详细统计
- 使用率阈值告警（>80%高使用率，>60%中高使用率）
- 系统报告生成和导出

## 系统要求

- Windows 10/11
- Python 3.8+
- PyQt6 6.10.1
